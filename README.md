# DeepSleep Blog Frontend

React + Vite frontend for DeepSleep Blog.

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Configure [.env.local](.env.local) with your backend API base URL:
   `VITE_API_BASE_URL="http://localhost:8080/api"`
3. Run the app:
   `npm run dev`
